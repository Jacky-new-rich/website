VI-LINK-TB RS485 端子台

可搭配感測器：適用於VI-TEX全系列

    • VI-TEX的標準轉接座，能夠以接線的方式讓感測器直接與PLC連接，共有八條端口，包含GND、Power、Out、In、Tx+、Tx-、Rx+、Rx-。
    • Shell MaterialAluminum
    • Operating Temperature Range-40℃ ≦ T ≦ 85℃
    • Input10 Pin Shielded Connector
    • I/O Portup to 8 Port
    • DC Input12 ~ 24 V / 200mA
    • Weight60g

VI-LINK-EXIO 端子台

可搭配感測器：適用於VI-TEX全系列

    • 能夠以接線的方式讓感測器直接與PLC連接，且提供更多的IO輸出輸入接口，能夠擁有自由度更高的閾值觸發方式。
    • Shell MaterialAluminum
    • Operating Temperature Range-40℃ ≦ T ≦ 85℃
    • I/O PortIN*7、NO*7
    • No24V / 1.5A
    • Input5 ~ 24V
    • Weight100g

VI-LINK-USB 485 轉 USB 轉接座

可搭配感測器：適用於VI-TEX全系列

    • 由Virtual Com port的方式來將訊號傳輸至電腦，並且由USB接口來提供感測器電源。除了能夠觀看及儲存即時數據，也能協助您進行修改VI-TEX的標準震動閾值或是滿足其他自行開發的需求。
    • Shell MaterialAluminum
    • Operating Temperature Range-40℃ ≦ T ≦ 85℃
    • Input10 Pin Shielded Connector
    • OutputMicro USB
    • DC Input5 V / 500mA
    • Weight60g

VI-LINL-BT 485 轉藍芽轉接座

可搭配感測器：適用於VI-TEX全系列

    • 支援BT 5.0通訊，將VI-TEX的訊號直接傳輸至電腦，最遠傳輸距離可超過200米，並支援NFC近場感應快速配對設定，也可支援各式天線，用於增加無線傳輸穩定性。且內建電池，可以在不輸入額外電源的狀態下持續使用8小時，機動性較高。
    • Shell MaterialAluminum
    • Operating Temperature Range-40℃ ≦ T ≦ 85℃
    • Weight60g

VI-LINK-WIFI 485 轉WiFi轉接座

可搭配感測器：適用於VI-TEX全系列

    • 可透過WiFi接收器，將VI-TEX的訊號直接傳輸至電腦，需要額外連接電源，機動性較藍芽版本受限，但能夠支援的無線傳輸範圍更大。
    • Shell MaterialAluminum
    • Operating Temperature Range-40℃ ≦ T ≦ 85℃
    • Weight60g
