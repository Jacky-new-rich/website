Description of Application

Which kind of VI-TEX sensor is suitable for me?

VI-TEX has different versions for single axis and triaxial. You can following below feature information to choose the suitable one:

    • Single Axis
    • 1. The object is usually in a specific unidirectional vibration.
    • 2. Only need low data transformation, or low data wireless transformation.
    • 3. Already have enough knowledge about your equipment and have a fixed monitoring target.
    • 4. Example: Rail vehicles, elevators, floor rigidity, etc.
    • Triaxial
    • 1. The object usually has different characteristics vibration in multi-directional.
    • 2. Need to closely monitore with all variables in the environment.
    • 3. Not sure the monitoring target and want to build up a new vibration Algorithm.
    • 4. Example: Smart grinder, CNC, high-precision machine, wafer processing, etc.

What signals does VI-TEX monitor?

  • 1. A specific directions of vibration, can support up to three-axis.(horizontal (front and back, left and right) and vertical (up and down)).

  • 2. A variety of sensitivity options. Although in different fields, it also can capture the vibration signal from small to large.

  • 3. Support the fuction for converting signal characteristics such as displacement, velocity or acceleration. Since vibration is composed by a multiple complex frequency, frequency, amplitude and phase can all be used for trigger settings. 。

  • Signal Description
    Speed It's suitable for checking the condition of rotating machinery. The frequency band is from 10 to 1000 Hz, which is the most used unit of measurement.
    Acceleration It's used to measure higher frequency vibrations such as bearings or gearboxes. It is suitable for monitoring vibrations from generate pulses, like bearing wear, scratches or debris.
    Displacement Indicates the fluctuation range of vibration. It's suitable for measuring unbalance vibration like fans, blowers, etc. The frequency band is very low, the range is from 10 Hz to hundreds of Hz.
  • ４. Support multiple application fields, especially for long-term monitoring, such as: aging rate of machinery, internal parts abnormal detection, base aging, production line stability, production management, etc.

The limitation for VI-TEX

  • The object doesn't have an installation plat.

    If the size is too small or have no installation platform, we can not installed. But if the object is already fixed on another platform, we acan also install it on the platform. Or choosing a customized installation base.

  • Object has no fixed operating mode.

    Machines without a fixed operating mode cannot establish the trigger values or provide predictive indicators through vibration detection. If you need to measeue it, it need to use a long time and observe the short, medium and long term aging curve to provide special services.

  • Only provide the predictors.

    VI-TEX can provide indicators of current mechanical operating conditions and health assessments. If you need to achieve anti-vibration effects, you need to purchase other anti-vibration equipment.

  • Limited using time in no power environment

    VI-TEX is designed for long-term monitoring and providing high-speed edge computing capabilities. It can greatly save the burden of other equipment. Therefore, it must be connected to the power and the battery is only support special needs.
