Description

VI-TEX-LITE is an entry model for the vibration detecting for single axis. It keep the function for fornt-end signal processing, which can ensure the data quality and not easily affected by the environment. VI-TEX-LITE can also provide you trigger data to research or develop applications. If this is the first time you want to do the vibration detecting, you can start with VI-TEX-LITE will be easier. But please notice that it doesn't provide the service for raw data output, thanks.
