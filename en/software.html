VI Analyzer

Help you to visualize the data to simply the process for setting, trigger value adjustment. Provide you more real-time analysis and offline report functions.

Description

By using VI-LINK-USB, you can connect VI-TEX to PC and use VI-Analyzer to set trigger conditions, and view and save real-time data. It can provide variety monitoring service in different environments based on the category you want to detect. You can also use VI-Analyzer to read historical data for signal re-analysis, event marking and other applications. For detailed functions of it, please refer to the "Software Manual".
